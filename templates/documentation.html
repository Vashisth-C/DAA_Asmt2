DYNAMIC PROGRAMMING ALGORITHM

FUNCTIONS


1. solve(string rna, vector>& traceback):

Input:

  • rna: A string representing a sequence of RNA nucleotides.
  • traceback: A 2D vector representing the traceback matrix to store split points.

Output:

  • Returns an integer representing the maximum number of base pairs in the optimal secondary structure.

Mechanism of the function:

  • Initializes a matrix OPT to store optimal solutions for subproblems.
  • Iterates over possible lengths of subsequences (k) from 4 to n, where n is the length of the RNA sequence.
  • For each subsequence length k, iterates over all possible starting indices i and calculates the corresponding ending index j.
  • Considers two cases:
    • Case 1: No base pair is formed at position j, so the maximum number of pairs remains the same as the previous position.
    • Case 2: Checks if a base pair can be formed between positions t and j. If yes, calculates the number of pairs formed by including this base pair.
  • Updates the OPT matrix with the maximum number of pairs for the current subsequence length k.
  • Updates the traceback matrix with the split point for the current subsequence.
  • Returns the maximum number of pairs for the entire RNA sequence.

2. tracebackSecondaryStructure(string rna, vector > traceback, string & structure)

Input:

  • rna: A string representing a sequence of RNA nucleotides.
  • traceback: A 2D vector representing the traceback matrix with split points.
  • structure: A string to store the secondary structure of the RNA sequence.

Output:

  • Returns a vector of pairs representing the indices of base pairs in the secondary structure.

Mechanism of the function:

  • Initializes a stack to store the split points for the secondary structure.
  • Recursively constructs the secondary structure by splitting the RNA sequence based on the split points stored in the traceback matrix.
  • Updates the structure string with the base pairs formed in the secondary structure.
  • Returns a vector of pairs representing the indices of base pairs in the secondary structure.
